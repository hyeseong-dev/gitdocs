# 등록된 저장소 지우기

저장소를 지우고 싶으면 어떻게 할까요?

```text
# 저장소 입력 명령어 
$ git remote add origin 경로주소 

# 등록된 저장소 확인 
$ git remote -v
origin  C:/Users/?댄삙??Desktop/ (fetch)
origin  C:/Users/?댄삙??Desktop/ (push)

# 등록된 저장소 삭제 
$ git remote remove 저장소이름(origin)
```

위 명령어를 실행하면 아래와 같이 실제 터미널에서 확인할 수 있어요.
