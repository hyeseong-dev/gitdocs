# 백그라운드

## 명령어 백그라운드로 실행하기

$ python minicafe.py &

## 명령어 백그라운드로 실행하기\(콘솔창 닫아도 실행됨\) - 참고로 콘솔 로그는 nohup.out에 쌓임.

$ nohup python minicafe.py &

## 실행중인 프로세스 모두 보기\(검색으로 찾기 : ps -ef \| grep 검색어\)

$ ps -ef

## 실행중인 프로세스 종료하기\(백그라운드 실행 포함\)

$ kill -9 \[PID\]
