위의 6개 명령어에 대해 학습해 볼게요.

### &gt; mkdir \[옵션\] 디렉토리

### 옵션 파라미터

```text
-m --mode=mode : set permissions    
-p --parents    : 경로 이름이 될수 있는데요. 만약 경로내에 이름이 존재하지 않으면 자동으로 해당 디렉토리를 생성해줘요.  
-v verboss      : 새로운 디렉토리 생성시 정보가 나타나요.   
```
