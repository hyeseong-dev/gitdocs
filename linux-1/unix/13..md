# 13. 디렉토리와 파일 생성

## 디렉토리 생성 

 make directory의 약자가 mkdir이에요. 

#### 현재 경로에서 디렉토리 만들기 

> mkdir  디렉토리명

#### 다른 경로에서 디렉토리 만들기 

> mkdir parent/child 또는   
> mkdir ./child

## 파일 생성 



#### 현재 경로에서 file 만들기 

> touch 파일명

#### 다른 경로에서 file 만들기 

> touch parent/child 또는   
> touch  ./child
